# 🚀 FinLedger Pro – Backend Banking Infrastructure

## 📌 Overview
FinLedger Pro is a production-style backend banking system built using Node.js and Express. It simulates real-world financial operations such as authentication, account management, and transaction processing using a double-entry ledger system to ensure accuracy and consistency.

# Features

- 🔐 JWT-based Authentication & Authorization  
- 🏦 Bank Account Management  
- 💸 Credit & Debit Transactions  
- 📊 Double-Entry Ledger System  
- 🧾 Transaction History Tracking  
- 📧 Email Notifications (Nodemailer)  
- ⚡ RESTful API Architecture  


## 🛠️ Tech Stack

- **Backend:** Node.js, Express.js  
- **Database:** MongoDB (Mongoose)  
- **Authentication:** JWT  
- **Other Tools:** Nodemailer, Dotenv, Cookie-Parser
